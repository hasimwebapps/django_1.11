# Setup
## Setup Local Env
- python3 -m venv Env
- source Env/bin/activate

## Upgrade pip
- pip install --upgrade pip

## Check pip list & version
- pip list

## install Django 1.11
- pip install Django==1.11.17
- pip list

# Start Project
## Start Project
- django-admin startproject mywebsite

## Run Project
-  python manage.py runserver
